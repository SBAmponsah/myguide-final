# MyGuide

MyGuide is a client-side academic planning web application designed to help students organize their courses, schedules, and academic tasks in one place.

## Overview
This project was developed as part of a web application development course. The goal was to build a functional, multi-page client-side application using core web technologies while focusing on usability and clean project structure.

## Features
- Multi-page interface (Dashboard, Calendar, Courses, Profile, Help)
- Dynamic content updates using JavaScript
- Persistent data storage using browser localStorage
- Organized file structure for scalability
- Deployed using GitHub Pages

## Technologies Used
- HTML
- CSS
- JavaScript
- GitHub Pages
